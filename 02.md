```
if(st.contains(str[i]))
    st = st.replace(str[i], "!")
```
- str[i]를 !로 치환한다는 
- contains()메소드는 문자열 내에 존재하는 특정문자 시퀀스를 확인합니다.
- 문자열 치환 함수 replace, replaceAll, replaceFirst

### replace
```
String a = "무궁화. 삼천리. 화려강산. 대한사람. 대한으로. 길이. 보전하세 ";
//replace([기존문자],[바꿀문자])
a = a.replace(".", "/");
System.out.println(a);

//결과값 : 무궁화/ 삼천리/ 화려강산/ 대한사람/ 대한으로/ 길이/ 보전하세
```

### replaceAll
```
String a = "무궁화. 삼천리. 화려강산. 대한사람. 대한으로. 길이. 보전하세 ";
//replaceAll([정규식],[바꿀문자])
a = a.replaceAll(".", "/");
System.out.println(a);

//결과값 : /////////////////////////////////////
```
